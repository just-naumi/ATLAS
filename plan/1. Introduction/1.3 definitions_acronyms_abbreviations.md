# Perencanaan Penulisan SRS - 1.3 Definitions, Acronyms, and Abbreviations

Dokumen ini berisi rencana penulisan terfokus untuk bagian **1.3 Definitions, Acronyms, and Abbreviations** dari dokumen *Software Requirements Specification* (SRS) untuk sistem **Decentralized P2P Train Telemetry & Braking Alert System**.

## Fokus Isi Bagian 1.3 Definitions, Acronyms, and Abbreviations

Pada bagian ini, kita akan mendefinisikan semua istilah teknis, akronim, dan singkatan yang digunakan dalam dokumen SRS ini agar seluruh pembaca — baik teknis maupun non-teknis — memiliki pemahaman yang seragam. Poin-poin yang akan dimasukkan ke dalam draf akhir adalah:

1. **Definisi Istilah Teknis:**
   - **Decentralized System:** Sistem yang beroperasi tanpa pusat kendali tunggal; setiap node memiliki kemampuan pemrosesan dan pengambilan keputusan secara mandiri.
   - **P2P (Peer-to-Peer):** Arsitektur komunikasi di mana setiap perangkat (node) dapat berkomunikasi langsung dengan perangkat lain tanpa melalui server perantara.
   - **Telemetry:** Proses pengukuran dan transmisi data dari jarak jauh secara otomatis; dalam konteks ini, mengacu pada data posisi dan kecepatan kereta yang dikirimkan antar-unit.
   - **Braking Alert:** Sinyal peringatan yang dihasilkan sistem ketika terdeteksi risiko tabrakan, yang dapat bersifat informatif (untuk masinis) atau otomatis (mengaktifkan rem).
   - **Blind Spot:** Kondisi di mana dua kereta tidak dapat saling melihat secara langsung (misalnya di tikungan atau terowongan), sehingga deteksi via GPS dan komunikasi radio menjadi kritis.
   - **Fail-Safe:** Prinsip desain sistem di mana kegagalan suatu komponen akan membawa sistem ke kondisi yang aman (misalnya, mengaktifkan pengereman darurat secara otomatis).
   - **On-Board Processing:** Pemrosesan data yang dilakukan langsung di dalam unit perangkat keras yang terpasang di kereta, tanpa perlu mengirimkan data ke server eksternal.

2. **Akronim (Acronyms):**

   | Akronim | Kepanjangan |
   |---------|-------------|
   | SRS     | Software Requirements Specification |
   | ATLAS   | Autonomous Train Localization & Alert System |
   | LoRa    | Long Range (protokol komunikasi radio jarak jauh) |
   | GPS     | Global Positioning System |
   | RTK     | Real-Time Kinematic (teknik GPS presisi tinggi) |
   | P2P     | Peer-to-Peer |
   | MCU     | Microcontroller Unit |
   | OCC     | Operations Control Center |
   | IEEE    | Institute of Electrical and Electronics Engineers |
   | API     | Application Programming Interface |
   | LED     | Light-Emitting Diode |
   | LCD     | Liquid Crystal Display |
   | RSSI    | Received Signal Strength Indicator |

3. **Singkatan (Abbreviations):**
   - **ms:** milidetik (millisecond)
   - **km/h:** kilometer per jam
   - **m:** meter
   - **Hz:** Hertz (satuan frekuensi)
   - **dBm:** desibel-miliwatt (satuan kekuatan sinyal)
