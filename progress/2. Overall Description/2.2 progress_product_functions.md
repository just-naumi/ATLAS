# 2. Overall Description

## 2.2 Product Functions

Sistem ATLAS ini dirancang dan dikembangkan untuk dapat melakukan beberapa fungsi diantaranya adalah sebagai berikut:
a. Pemosisian presisi geografis di mana sistem dapat mengekstrak data koordinat seperti lintang dan bujur serta kecepatan laju lokomotif secara real-time melalui pemrosesan modul satelit GPS RTK.
b. Transmisi data terdesentralisasi di mana sistem dapat menyiarkan broadcasting dan menerima paket data telemetri antar kereta secara langsung dengan memanfaatkan komunikasi frekuensi radio LoRa P2P.
c. Kalkulasi ancaman tabrakan di mana mikrokontroler dapat memproses perhitungan jarak relatif antar kereta secara lokal untuk mendeteksi potensi terjadinya tabrakan.
d. Peringatan dini di mana sistem dapat mengaktifkan instrumen indikator visual dengan lampu LED dan layar serta membunyikan alarm suara di dalam kabin masinis apabila jarak aman telah terlampaui.
e. Intervensi pengereman otomatis di mana sistem dapat memicu perangkat relai aktuator untuk memutus laju kendaraan secara paksa apabila tidak mendeteksi adanya respons pengereman manual dari masinis dalam interval waktu tertentu.
