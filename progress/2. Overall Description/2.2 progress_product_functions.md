# 2. Overall Description

## 2.2 Product Functions

Sistem ATLAS ini dirancang untuk dapat melakukan beberapa fungsi diantaranya adalah sebagai berikut:
1) Pemosisian Presisi Geografis, di mana sistem dapat mengekstrak data koordinat (lintang dan bujur) serta kecepatan laju lokomotif secara *real-time* melalui pemrosesan modul satelit GPS RTK.
2) Transmisi Data Terdesentralisasi, di mana sistem dapat menyiarkan (*broadcasting*) dan menerima paket data telemetri antar-kereta secara langsung memanfaatkan komunikasi frekuensi radio LoRa P2P.
3) Kalkulasi Ancaman Tabrakan, di mana mikrokontroler dapat memproses perhitungan jarak relatif antar-kereta secara lokal untuk mendeteksi potensi terjadinya tabrakan.
4) Peringatan Dini, di mana sistem dapat mengaktifkan instrumen indikator visual (lampu LED/layar) serta membunyikan alarm suara di dalam kabin masinis apabila jarak aman telah terlampaui.
5) Intervensi Pengereman Otomatis, di mana sistem dapat memicu perangkat relai aktuator untuk memutus laju kendaraan secara paksa apabila tidak mendeteksi adanya respons pengereman manual dari masinis dalam interval waktu tertentu.
