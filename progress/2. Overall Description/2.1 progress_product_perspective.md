# 2. Overall Description

## 2.1 Product Perspective

Sistem ATLAS merupakan sistem tertanam tambahan yang dirancang sebagai pengaman sekunder untuk mendeteksi jarak aman serta mencegah potensi tabrakan antar kereta api terutama pada area blind spot masinis. Dalam pengoperasiannya, sistem ini melibatkan masinis sebagai pengguna utama di lokomotif, serta teknisi dan tim pengembang sebagai pihak yang bertanggung jawab terhadap pengelolaan dan pemeliharaan perangkat. Sistem keselamatan berbasis desentralisasi ini memanfaatkan teknologi mikrokontroler, modul komunikasi radio LoRa, sensor GPS RTK, serta modul relay mekanik untuk mendukung proses pengembangan perangkat keras dan lunak. Dengan pendekatan tersebut, sistem dapat beroperasi tanpa ketergantungan pada infrastruktur server pusat maupun jaringan seluler.

Sebagai perangkat yang bersifat stand-alone, sistem ATLAS berinteraksi dengan lingkungannya melalui beberapa antarmuka utama. Pada sisi komunikasi, sistem memanfaatkan mekanisme P2P berbasis pancaran sinyal LoRa untuk melakukan pertukaran data antar kereta secara langsung sekaligus menerima pembaruan data koreksi (RTCM) dari Base Station GPS RTK guna meningkatkan akurasi posisi. Pada sisi perangkat keras, sistem menggunakan mikrokontroler yang terintegrasi dengan sensor input berupa GPS RTK serta perangkat aktuator output seperti relay untuk keperluan pengereman darurat, dan indikator peringatan untuk masinis berupa buzzer maupun LED. Selain itu, antarmuka pengguna dirancang dalam bentuk instrumen peringatan visual dan audio yang sederhana di dalam kabin lokomotif agar masinis dapat merespons potensi bahaya dengan cepat tanpa mengurangi fokus selama pengoperasian kereta.
